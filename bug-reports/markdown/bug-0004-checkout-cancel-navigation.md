# Bug Report – “Cancel” button fails to return to cart page

- **ID:** BUG-0004
- **Title:** Cancel button does not redirect user back to the cart
- **Severity:** Low
- **Priority:** Medium
- **Environment:** Chrome 140.0.7339.208, Windows 11
- **Preconditions:** User is on the checkout information page

## Steps to Reproduce

1. Login to https://www.saucedemo.com/
2. Add a product to the cart
3. Proceed to checkout
4. Click the “Cancel” button

## Expected Result

User should be redirected back to the cart page.

## Actual Result

The user returns to mainpage

## Impact

Minor UX issue — may confuse users who want to cancel the checkout process.
